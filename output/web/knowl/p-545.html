Se \(f\colon A\subseteq\R^3\to\R\) é uma função diferenciável e \(\gamma\colon I\subseteq\R\to\R^3\) é uma curva derivável tal que a imagem de \(\gamma\) está contida na superfície de nível \(c\) de \(f\text{,}\) então \(f\big(\gamma(t)\big)=c\text{,}\) para todo \(t\in I\text{.}\) Utilizando uma generalização (relativamente óbvia) do Teorema A.6.4 para o \(\R^3\) , temos que
\begin{equation*} 0=(f\circ\gamma)'(t)=\Big\langle\nabla f\big(\gamma(t)\big),\gamma'(t)\Big\rangle. \end{equation*}
