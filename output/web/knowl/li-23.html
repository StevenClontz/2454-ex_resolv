Item 1.3.7.d.
Dizer que a imagem de uma curva, \(\eta(t)=\big(x(t), y(t), z(t)\big)\) está contida no gráfico de uma função \(f\colon A\subseteq\R^2\to\R\text{,}\) significa exatamente dizer que \(z(t)=f\big(x(t),y(t)\big)\text{.}\)
No caso deste exercício isso significa que \(z(t)=f(\sin t, \cos t)=\dfrac{2\sin^2t+4\cos^2t}{\sin^2t+\cos^2t+1}=1+\cos^2t \text{.}\)Com isso, \(\Gamma(t)=(\sin t,\cos t, 1+\cos^2t)\) e, consequente \(\Gamma'(t)=(\cos t, -\sin t, -2\cos t\sin t)\implies \Gamma'(t)=\big(\dfrac{1}{2},-\dfrac{\sqrt{3}}{2},-\dfrac{\sqrt{3}}{2}\big).\)
