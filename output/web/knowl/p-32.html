Caso precise, recorde o que é uma curva de nível e veja alguns exemplos na Seção A.2. Uma dica para cada item:
  1. Depois de escrever a equação que define cada curva de nível, é um exercício de reconhecer cônicas a partir de suas equações reduzidas.
  2. É o clássico caso de parametrizar uma cônica
  3. Vetor tangente num ponto? É só derivar a parametrização!
  4. Basta lembrar que se \((x,y,z)\in Gr f\text{,}\) então \(z=f(x,y)\text{.}\) Para nossos propósitos, uma curva em \(\R^3\) tem as mesmas características e propriedades (continuidade, derivabilidade, etc.) de uma curva no plano, mas "com uma coordenada a mais".
