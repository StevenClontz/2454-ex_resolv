Observe que nem sempre uma curva de nível de uma função \(F\colon A\subseteq\R^2\to\R\) é o gráfico de função \(f\colon I\subseteq\R\to\R\text{.}\) O mesmo ocorre para superfícies de nível de funções a três variáveis: a esfera acima é uma superfície de nível que não é gráfico de uma função a duas variáveis (por quê?).
